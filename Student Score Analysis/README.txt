# 📊 Student Score Analysis

This project performs exploratory data analysis (EDA) on a dataset of student scores. Using Python and Jupyter Notebook, the notebook demonstrates how basic statistical tools and visualizations can be used to understand patterns in study habits and performance.

## 📁 Dataset

The project uses a file named `student_scores.csv`, which contains:

- `Name`: Name of the Student
- `Scores`: Score achieved in Maths, Science and English

## 🧰 Tools Used

- Python 3
- pandas
- numpy
- matplotlib

## 📌 Features

- Load and inspect a real-world dataset
- Perform basic data analysis and summary statistics
- Plot data using line graphs and scatter plots


## 🚀 How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/student-score-analysis.git
   cd student-score-analysis

2. Install dependencies:
   - pip install pandas numpy matplotlib

3. Run the notebook:
   - jupyter notebook main.ipynb
